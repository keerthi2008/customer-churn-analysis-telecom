# customer-churn-analysis-telecom

# 📊 Customer Churn Analysis (Telecom Project)

## 🔷 Project Overview

This project focuses on analyzing customer churn in a telecom company using Excel .

The goal is to identify key factors that influence customer churn and provide actionable insights to improve customer retention.

---

## 🎯 Objectives

- Analyze customer behavior and churn patterns
  
- Identify high-risk customers
  
- Build visual dashboard for decision-making
  
- Provide business recommendations  

---

## 📂 Dataset Information

The dataset contains customer details such as:

- Customer ID
  
- Gender
  
- Senior Citizen
  
- Partner & Dependents
  
- Tenure
  
- Services (TechSupport, OnlineSecurity, etc.)
  
- Contract Type
  
- Payment Method
  
- Monthly Charges
  
- Total Charges
  
- Churn (Target Variable)  

---

## 🧹 Data Cleaning

- Removed duplicates
  
- Handled missing values in `TotalCharges`
  
- Standardized Yes/No values
  
- Converted categorical data into numerical format  

---

## 📊 Exploratory Data Analysis (EDA)

Performed analysis using Pivot Tables and Charts:

- Churn Distribution
  
- Churn by Gender
  
- Churn by Contract Type
  
- Monthly Charges vs Churn
  
- Tenure vs Churn
  
- Services Impact on Churn  

---

## 📌 Key Insights

- Customers with **month-to-month contracts** have higher churn
  
- High **monthly charges** increase churn probability
  
- Customers with **low tenure** are more likely to leave
  
- Lack of **TechSupport and OnlineSecurity** increases churn  

---

## 📈 KPI Metrics

- Total Customers
  
- Total Churn Customers
  
- Churn Rate (%)
  
- Active Customers
  
- Average Monthly Charges

- Average Tenure  

---

## 📊 Dashboard

An interactive Excel dashboard was created with:

- KPI cards
  
- Charts and visualizations
  
- Slicers for filtering  

---

## 🤖 Tools Used

- Microsoft Excel
   
- Data Visualization  

---

## 💡 Recommendations

- Offer discounts for high-charge customers
  
- Encourage long-term contracts
  
- Improve customer support services
  
- Target new customers with retention strategies  

---

## 🚀 Conclusion

This project helps understand customer churn behavior and provides data-driven strategies to improve customer retention.
